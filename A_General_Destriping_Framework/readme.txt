This is code of ``A General Destriping Framework for Remote Sensing Images Using Flatness Constraint,''.

1) Setting parameters
 - Choose the type of a target image (target_image)
 - Adjust the balancing parameter (para.lambda_S).
 - Set as use_GPU = 1 if you use GPU.
 - Select an image regularization (para.regularization).

2) Run main.m

For more details, see [1]

[1] K. Naganuma, S. Ono, ``A General Destriping Framework for Remote Sensing Images Using Flatness Constraint,'' IEEE Trans. Geosci. Remote Sens., 2022.
